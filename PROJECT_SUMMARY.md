# 基于VSCode社区版的简易DBMS全栈开发项目总结

## 项目概述

本项目是一个基于VSCode社区版的轻量级数据库管理系统(DBMS)全栈开发项目。项目采用现代化的技术栈，实现了类似MySQL的核心功能，包括完整的SQL解析引擎、高效的存储引擎和直观的Web管理界面。

## 技术架构

### 核心技术栈
- **后端**: Node.js + TypeScript + Express.js
- **前端**: React 18 + TypeScript + Tailwind CSS
- **数据库引擎**: 自研轻量级DBMS
- **SQL解析**: ANTLR4语法解析器
- **开发工具**: VSCode + ESLint + Prettier + Jest

### 五层架构设计
1. **网络接口层**: RESTful API + WebSocket实时通信
2. **SQL接口层**: 语法解析、查询优化、执行计划
3. **查询执行层**: 事务管理、并发控制、缓存管理
4. **存储引擎层**: 页面管理、索引管理、记录管理
5. **文件系统层**: 数据文件、日志文件、配置文件
